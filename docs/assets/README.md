# Assets

Images and diagrams used by the documentation pages.

Conventions:

- File names: `kebab-case.png` / `.svg` / `.jpg`.
- Product screenshots are prefixed with the product name (`tiger-studio-inventory.png`).
- Prefer Mermaid blocks inside the Markdown pages for diagrams; only export a
  static image when Mermaid cannot express it.
